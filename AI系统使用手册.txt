📊 系统操作说明
=====================================

💻 【日常操作流程】
1. 运行演示系统 → 获取AI建议
2. 查看预测收益和风险评分
3. 根据AI建议执行交易决策
4. 监控系统运行状态

🎯 【关键指标解读】

📈 预测收益:
  • > 3%: 强烈推荐
  • 1-3%: 建议执行  
  • 0-1%: 谨慎操作
  • < 0%: 避免交易

⚠️ 风险评分:
  • 0-0.3: 低风险 ✅
  • 0.3-0.6: 中等风险 ⚡
  • 0.6-1.0: 高风险 🚨

🎪 资金配置:
  • 显示各策略最优分配比例
  • 基于风险调整收益计算
  • 自动满足配置约束

🔧 【系统定制】

修改风险参数:
```python
# 在ai_system_demo.py中调整
risk_threshold = 0.6  # 风险预警阈值
min_allocation = 0.05  # 最小配置比例
max_allocation = 0.50  # 最大配置比例
```

添加新策略:
```python
# 扩展strategies列表
strategies = [
    ('新策略名称', '交易对'),
    # ... 其他策略
]
```

💡 【使用建议】
=====================================

✅ 最佳实践:
  • 每日运行一次获取最新AI建议
  • 关注风险预警信号
  • 根据市场变化调整参数
  • 定期检查模型表现

⚠️ 注意事项:
  • AI建议仅供参考，请结合实际市场判断
  • 高风险时期应降低仓位
  • 定期备份交易记录
  • 保持资金管理纪律
